# Portfolio Website

A modern portfolio website built with React to showcase projects and skills.

## Features

- Responsive design
- Project showcase
- About section
- Contact information
- Smooth navigation

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

The website will open at `http://localhost:3000`

## Project Structure

```
src/
├── components/
├── pages/
├── styles/
└── App.js
```

## Technologies Used

- React
- CSS3
- JavaScript ES6+
